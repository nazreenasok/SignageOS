Deploy to TVs
  1. 1Run: python main.py — server on port 8000
  2. 2Find your PC's local IP: open CMD and run ipconfig
  3. 3On each TV browser open: http://YOUR_IP:8000/display
  4. 4Each TV auto-registers in the Screens tab — assign playlists there
  5. 5Enable kiosk mode in TV browser settings to hide browser UI
